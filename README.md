# Camera-Sales-Dashboard---Excel
Project Overview

This project showcases an interactive Camera Sales Dashboard built in Microsoft Excel. The dashboard provides a comprehensive view of sales performance through dynamic charts, KPIs, slicers, and business insights.

It enables users to monitor sales trends, profitability, buyer behavior, and geographical performance, supporting better business decision-making.

Dashboard Features

-  KPI Cards
- Total Cost
- Gross Sales
- Net Sales
- Countries Covered

-  Profit Analysis by Product
-  Units Sold by Buyer Type
-  Country-wise Sales Distribution
-  Monthly Sales Trend Analysis
-  Interactive Filters (Buyer Type & Discount Band)
-  AI-Powered Key Influencers Analysis to identify the factors affecting profitability
